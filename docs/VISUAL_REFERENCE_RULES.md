# Red Hollow — Visual Reference Rules

Regras para uso de imagens de referência, moodboards e inspiração externa.

## Princípio

Referências servem para **tom, iluminação, composição e qualidade** — nunca para cópia literal.

Toda arte final de Red Hollow deve ser **original**.

## O que é permitido extrair de moodboards

- atmosfera geral (decadência, terror religioso, contraste de luz);
- densidade de pixel art desejada;
- sensação de profundidade em cenários 2D;
- ritmo de animação (agressivo, legível);
- proporção corporal em estilo anime aplicada a sprites;
- uso **moderado** de vermelho como acento.

## O que é proibido copiar

- personagens específicos;
- símbolos ou logotipos;
- layouts de interface;
- mapas ou geometria de nível;
- poses únicas reconhecíveis;
- designs de inimigos;
- cenários completos;
- qualquer elemento identificável de IP de terceiros.

## Processo recomendado

1. Coletar referências em pasta interna **não versionada** ou em ferramenta de arte (não commitar assets de terceiros sem licença).
2. Escrever notas: “o que queremos” vs “o que não queremos”.
3. Produzir thumbnails originais antes de pixel art final.
4. Revisar contra esta lista antes de integrar no repositório.

## Vermilite e referências

Se uma referência usar vermelho dominante, **não** reproduzir essa saturação no jogo inteiro. Extrair apenas a ideia de “acento perigoso” e aplicar conforme `ART_BIBLE.md`.
